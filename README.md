README
======

Install
-------

```
$ sudo apt install virtualenv
```

```
$ mkvirtualenv --python=/usr/bin/python3 matchplay-wallscreen
(matchplay-wallscreen) $ pip3 install flask
(matchplay-wallscreen) $ pip3 install requests
```

Running
-------

Edit `config.py`

To run:

```
$ workon matchplay-wallscreen
$ ./app.py
```
